# Aiddgen Flashcard App

A web-based flashcard app using FSRS algorithm to learn how aiddgen/ works.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

```
flashcard-app/
├── src/
│   ├── components/    # React components
│   ├── utils/         # Utility functions
```
